# Flipkart E-Commerce Data Analysis using Power BI

## Project Overview
This project focuses on analyzing Flipkart e-commerce product data using Microsoft Power BI. The main objective of this project is to explore product information such as category, price, discount, brand, and ratings to gain useful insights through data visualization.

## Dataset
The dataset used in this project was collected from Kaggle. It contains information about Flipkart products including product name, category, brand, price, discount, and customer ratings. The dataset was cleaned and prepared before performing the analysis.

## Data Processing
The dataset was processed by removing missing values, duplicate records, and unnecessary columns. Data cleaning was performed to make the dataset suitable for analysis and visualization in Power BI.

## Dashboard
An interactive Power BI dashboard was created to visualize the data and generate insights. The dashboard includes different charts and visualizations such as:
- Category-wise product distribution
- Price vs Discount analysis
- Top brands analysis
- Product rating insights

## Tools Used
- Microsoft Power BI
- Microsoft Excel
- Kaggle Dataset
- GitHub

## Key Insights
- Identified the most popular product categories.
- Analyzed pricing and discount trends.
- Compared brand performance based on product availability.
- Visualized customer ratings across different products.

## Conclusion
The project demonstrates how data visualization tools like Power BI can help transform raw data into meaningful insights. The dashboard provides a clear understanding of product trends and helps in making better data-driven decisions.
